Discover Codenvy in 5 seconds

A developer workspace has been created. It is composed of cloned source code and project runtime.


-
Run the App
A machine is launched and projects' source code are mounted in.

A set of commands is configured for the project. Execute a command by selecting it and hitting the run button.


-
Use the App
The app’s URL will appear in the Console panel, once the application started, click to open your app in another tab.



Stop process when you’re done.


-
Edit the App
In the Project Explorer, navigate to source code.

Edit the code and save the file.
-
Commit the Changes
Select Git > Commit. Select the checkbox, add a comment and commit. If you have permissions, you can push your change on the remote repository, or issue a pull request.

Open Pull Request panel. Provide a branch name and add comment. Click on contibute button, if you have the permissions on the repository, a pull requested will be issued.

-
Extend the IDE
Add custom commands and extend your IDE.
